# Лабораторные работы по дисциплине «Программирование»

Репозиторий создан в рамках изучения дисциплины **«Программирование»** (язык Python)

## Цель репозитория:

Демонстрация освоения **базового и среднего уровня** языка Python через выполнение лабораторных заданий.

## Что внутри?

- **8 лабораторных работ** (***Rare***, ***Medium***, ***Well-done*** сложности)
- Чистый, структурированный и документированный код
- Подробные отчеты с выполненной работой внутри каждой лабораторной (цель, условие, использованные конструкции, пояснения, примеры вывода)
- Дополнительные материалы для подготовки к экзамену
- Шпаргалки, вопросы к экзамену, рекомендуемые ресурсы

## Список лабораторных работ:

| 2  | Построение графиков в Python                   | matplotlib: line plot, scatter, bar, subplot, настройка осей, легенда, сохранение в файл  |

| 4  | Рекурсия                                       | Рекурсивные функции, базовый случай, дерево рекурсии, хвостовая рекурсия, мемоизация     |
