# Полезные ресурсы по Python

## Книги

- **Марк Лутц — "Изучаем Python" (Learning Python)**  
  Классика, очень подробная и глубокая.  
  - [Learning Python. Том 1](./Books/Learning_Python_Tom_1.pdf) — основы языка, структуры данных, функции, ООП  
  - [Learning Python. Том 2](./Books/Learning_Python_Tom_1.pdf) — продвинутые темы, модули, метапрограммирование, итераторы/генераторы и др.

- **Allen B. Downey — "Think Python"**  
  Отличная книга для начинающих с акцентом на мышление программиста.  
  - [Think Python](./Books/Think_Python.pdf) — основы языка, структуры данных, функции, ООП
