# Требования для допуска к экзамену по дисциплине «Программирование»

Для получения **допуска** к экзамену необходимо выполнить **все** следующие условия:

- Посетить **не менее 70%** занятий  
  (учитываются как лекционные, так и лабораторные занятия)

- Выполнить **диагностическое тестирование** не менее чем на **70%**

- Сдать **100%** лабораторных работ **базовой сложности**  
  (все обязательные задания должны быть приняты преподавателем)

Выполнение этих требований гарантирует получение оценки **«удовлетворительно»** («зачтено» / 3 балла) по дисциплине.

## Для получения более высокой оценки («хорошо» / «отлично»)

Необходимо дополнительно выполнить хотя бы одно из следующих условий (чем больше — тем выше балл):

- Сдать **не менее 70%** лабораторных работ **повышенного уровня сложности**  
  (дополнительные / усложнённые задания в каждой лабораторной)

- Успешно ответить на **теоретические вопросы** из экзаменационного билета  
  (устный/письменный ответ на билет, включая вопросы по C и Python)
